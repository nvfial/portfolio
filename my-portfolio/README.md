# 🚀 高级 Vue 3 作品集网站

一个**现代化、交互丰富、功能完整**的 Vue 3 响应式作品集网站，包含所有你需要的功能，甚至更多！

## ✨ 核心功能

### 🎨 视觉与交互
- ✅ **3D 粒子背景动画** - 使用 @tsparticles/vue3 实现流畅的粒子效果
- ✅ **粒子开关控制** - 可随时开启/关闭粒子效果，节省性能
- ✅ **暗黑/亮色模式** - 平滑过渡，自动记忆用户偏好
- ✅ **鼠标跟随光晕** - 动态光晕效果，增强交互体验
- ✅ **视差滚动效果** - 页面滚动时的视差动画
- ✅ **打字动画** - 首页标题自动打字效果
- ✅ **3D 卡片翻转** - 项目卡片支持 3D 鼠标跟随效果
- ✅ **页面过渡动画** - 路由切换时的流畅过渡

### 🎯 页面功能

#### 首页 (HomeView)
- 动态粒子背景
- 打字动画展示技能
- 鼠标光晕效果
- 视差标题动画
- 滚动指示器
- GSAP 动画库集成

#### 项目页面 (ProjectsView)
- 项目分类过滤
- 3D 卡片翻转效果
- 项目标签展示
- GitHub 链接
- 响应式网格布局
- ScrollTrigger 滚动动画

#### 关于页面 (AboutView)
- 技能进度条动画
- 工作经历时间线
- 个人介绍
- 头像展示（带光晕效果）
- 技能雷达图样式

#### 联系页面 (ContactView)
- 表单验证（实时验证）
- 联系信息卡片
- 社交链接展示
- 表单提交动画
- 成功/错误状态提示

### 🎨 设计特性
- **玻璃态效果** (Glassmorphism) - 导航栏和按钮
- **渐变背景** - 多色渐变支持
- **流畅动画** - 使用 GSAP 和 CSS 动画
- **响应式设计** - 完美适配手机、平板、桌面
- **暗黑模式** - 完整的暗黑主题支持
- **CSS 变量** - 易于定制和主题切换

## 🛠️ 技术栈

- **Vue 3** - Composition API + `<script setup>`
- **Vite** - 极速构建工具
- **Vue Router** - 单页应用路由
- **@vueuse/core** - Vue 工具库（暗黑模式等）
- **@tsparticles/vue3** - 粒子动画库
- **GSAP** - 专业动画库
- **ScrollTrigger** - 滚动触发动画

## 🎨 自定义主题

所有颜色和样式都通过 CSS 变量定义，在 `src/assets/main.css` 中：

```css
:root {
  --bg-primary: #ffffff;
  --accent-primary: #667eea;
  /* ... 更多变量 */
}

.dark {
  --bg-primary: #0a0e27;
  --accent-primary: #818cf8;
  /* ... 暗黑模式变量 */
}
```

## 📝 项目结构

```
src/
├── assets/
│   └── main.css          # 全局样式（暗黑模式、CSS变量）
├── components/
│   ├── Navbar.vue       # 导航栏（玻璃态效果）
│   ├── DarkModeToggle.vue  # 暗黑模式切换
│   ├── ParticleToggle.vue  # 粒子开关
│   └── ProjectCard.vue   # 项目卡片（3D效果）
├── views/
│   ├── HomeView.vue     # 首页（粒子、打字、光晕）
│   ├── ProjectsView.vue # 项目页（过滤、3D卡片）
│   ├── AboutView.vue    # 关于页（技能、时间线）
│   └── ContactView.vue  # 联系页（表单验证）
├── router/
│   └── index.js         # 路由配置
├── App.vue              # 根组件（页面过渡）
└── main.js              # 入口文件
```
